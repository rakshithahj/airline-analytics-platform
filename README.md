
# Airline Intelligence Platform

A professional Streamlit analytics dashboard built using airline passenger data.

## Features

- Executive Dashboard
- Passenger Analytics
- Geographic Intelligence
- Pilot Performance
- AI Insights
- Interactive Plotly Visualizations

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Deployment

Push to GitHub and deploy using Streamlit Community Cloud.
